GitHub Pages Instructions
  • Local Changes: All content changes are saved to your browser's local storage.
  • Publishing: Use the "Export All Data" button to download updated JSON files, then commit and push them to your GitHub repository.
  • File Structure: Save the exported files in the data/ directory of your repository.
  • Auto-Deploy: GitHub Pages will automatically update your website when you push changes.
